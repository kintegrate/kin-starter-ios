# To use the app:

## Clone the repo:

```
git clone https://github.com/kintegrate/kin-starter-ios.git
cd kin-starter-ios
```

## Install dependencies:

`pod install`

## Run the project:

- Open the kin-starter-ios.xcworkspace file
- Click run in xcode to run
